Política de Privacidade – QConta

O aplicativo QConta respeita a privacidade dos usuários.

Coleta de dados

O aplicativo não coleta dados pessoais diretamente. No entanto, utiliza o Google AdMob, que pode coletar informações automaticamente.

Dados coletados por terceiros

O Google AdMob pode coletar dados como identificadores do dispositivo, dados de uso e localização aproximada.

Uso das informações

Esses dados são utilizados para exibir anúncios e melhorar a experiência do usuário.

Armazenamento

As informações do usuário são armazenadas apenas no dispositivo.
